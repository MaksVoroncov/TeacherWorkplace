## 🚀 Как запустить проект

### Требования:
- Visual Studio 2022 (Community или выше)
- .NET 8.0 SDK
- SQL Server LocalDB (входит в Visual Studio автоматически)

### Шаги:

1. **Откройте проект:**
   - Запустите Visual Studio 2022
   - File → Open → Project/Solution
   - Выберите `TeacherWorkplace.csproj`

2. **Восстановите NuGet-пакеты:**
   - Visual Studio сделает это автоматически при открытии
   - Или: Tools → NuGet Package Manager → Restore

3. **Запустите проект:**
   - Нажмите **F5** или кнопку ▶️ Run
   - База данных создастся автоматически при первом запуске

4. **Войдите в систему:**
   - Логин: `teacher`
   - Пароль: `teacher123`
